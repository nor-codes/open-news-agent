# Open News Agent

AI-powered news intelligence platform with agentic search and conversational insights, inspired by learning Agentic AI.  
Built for **learning and experimentation**, not commercial use.

---

## Overview

Open News Agent is a cloud-native platform that collects news articles from RSS feeds, indexes them into a search engine, and allows users to interact with the data using an **AI-powered conversational interface**.

The system uses an **agentic approach**, where an AI agent interprets user queries, retrieves relevant articles from a search engine, and generates contextual responses.

The project is designed primarily to **learn modern architecture patterns**, including:

- Microservices architecture
- Event-driven data pipelines
- Search platforms
- Agentic AI workflows
- Kubernetes deployments
- Observability and telemetry
- Infrastructure as Code

Initial data sources include RSS feeds such as:
https://rss.nytimes.com/services/xml/rss/nyt/World.xml
